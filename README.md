## Requirements
- Go installed
- NodeJS & npm installed
- MongoDB installed

## Setup
NPM:

`npm install -g webpack`

`npm install`

Install the Macaron web framework:

`go get gopkg.in/macaron.v1`

Add an empty directory in your project root called "data"

## Run
Run Webpack + Start Go Server:

`npm run dev`

Start MongoDB:

`npm run server`

Deploy/Build
`npm run build`
